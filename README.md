# dnsperfでの遊び方

## Install
```sh:Ubuntu
> sudo apt update
> sudo apt upgrade
> sudo apt install -y libssl-dev libldns-dev libck-dev libnghttp2-dev
> sudo apt install dnsperf
> sudo apt install gnuplot
```

## Download
ツール類をダウンロードして適当なフォルダに配置します。
```sh:Ubuntu
```

## Setting
```sh:Ubuntu
> chmod a+x dnsperf.sh
> chmod a+x resperf.sh
```
